# 💢 Colina Financial Management

A financial management system built with **Expo**, featuring dynamic SVG charts, **PostgreSQL** data storage, and optimized cascade entity cleanup operations.

---

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

- Node.js
- npm
- Expo CLI (optional)

### Install Dependencies

```bash
npm install
```

### Start the Application

```bash
npx expo start
```

After starting the development server, you can run the app on:

- Android Emulator
- iOS Simulator
- Expo Go
- Development Build

---

## 📁 Project Structure

This project uses Expo's file-based routing system.

```text
app/
├── (routes)
├── components
├── services
├── assets
└── database
```

Edit the files inside the `app` directory to begin development.

## 🛠️ Development

### Reset the Project

```bash
npm run reset-project
```

This command moves the starter template into the `app-example` directory and creates a fresh `app` folder.

### Linting

```bash
npx expo lint
```

### Testing

Add and configure Jest or your preferred testing framework as needed.
